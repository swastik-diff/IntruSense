## рҹҺҜ What Is IntruSense?

IntruSense is a **honeypot-first, AI-powered security platform** that turns your server into a trap.

Instead of building higher walls, we do the opposite: we plant **fake credentials, fake admin panels, fake database backups, and fake `.env` files** across your server. The moment an attacker touches one вҖ” we know. We profile them. We alert you in plain English.

No false positives. No alert fatigue. Just confirmed, real intrusions.

```
Traditional Security:  рҹҸ° Build walls вҶ’ Hope attackers bounce off
IntruSense:            рҹӘӨ Plant traps вҶ’ Catch attackers in the act
```

## рҹ”Ҙ Features

### рҹӘӨ Deception Layer
- **7 fake trap pages** вҖ” corporate login, WordPress admin, phpMyAdmin, .env, backup.zip, API endpoints
- **3 protocol honeypots** вҖ” HTTP, SSH, FTP running simultaneously
- **Psychological breadcrumb chain** вҖ” each page references the next, keeping attackers engaged

### рҹ§  Intelligence Engine
- **Threat Score 0вҖ“100** вҖ” weighted across: abuse history, attack patterns, Tor usage, datacenter origin, credential stuffing
- **15+ attack pattern classifiers** вҖ” SQL Injection, XSS, Directory Traversal, Brute Force, Canary Access, and more
- **Tool fingerprinting** вҖ” identifies Nmap, Masscan, curl, Python scripts, etc.
- **Local vs External** вҖ” detects LAN attackers and maps them to your city

### рҹ’¬ AI Plain-English Summaries
No security jargon. Just human-readable explanations:

| What happened | What IntruSense tells you |
|---|---|
| SQL Injection detected | *"Someone is trying to trick your database into revealing secret data by typing malicious commands into form fields."* |
| /.env accessed | *"A trap was triggered: someone accessed a fake decoy file we planted. This is a confirmed intruder already inside your system."* |
| Brute force SSH | *"Someone is rapidly guessing your SSH password using automated tools вҖ” tried 47 times in 60 seconds."* |
| Port scan detected | *"Someone is checking what network services your device has open вҖ” like rattling door handles before breaking in."* |

### рҹ“Ҡ Dashboard
- **Live world map** вҖ” attackers geolocated in real-time
- **Per-attacker profiles** вҖ” full timeline, credentials tried, tools used
- **Smart notifications** вҖ” grouped by IP + attack type to prevent alert fatigue
- **Attack feed** вҖ” scrolling live event stream with protocol, path, and classification
